# 🚀 프로젝트

🏨 **숙박 예약 사이트**

<br>

## 📖 프로젝트 설명
사용자가 사이트를 통하여 원하는 일자와 구성 조건으로 숙박을 예약하는 서비스입니다. <br>
관리자는 자신의 숙소를 관리하고, 매출 현황을 손쉽게 볼 수 있는 기능을 구현하였습니다.
